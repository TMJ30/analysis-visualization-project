# U.S. Metropolitan Real Estate Analysis 

## Overview 
This project is a collaborative group analysis focused on real estate market trends across major metroploitan areaas in the United States. Using Python-based data analysis and visualization techniques, the team aimed to identify meaningful patterns, relationships, and regional differences withing urban housing markets, and to communicate those insights through clear and effective visualizations.
